claw-squad helps you run several AI tools in one workflow. It is built for software tasks that need more than one model or agent at the same time.

Use it to:

- let different AI tools handle different steps
- keep track of what each worker is doing
- see progress in a tmux-based view
- manage software tasks with less switching between tools

It fits work like code review, feature planning, bug fixing, file edits, and test support.

## 🧩 How the workflow is organized

claw-squad works like a small team manager for AI tools.

A simple flow looks like this:

1. You give the app a task.
2. It breaks the task into parts.
3. Each part goes to the right AI worker.
4. The workers complete their steps in order.
5. You watch the process in the tmux view or other control screen.

This setup helps when one tool is better at planning, another is better at coding, and another is better at review.

## 🖥️ Using the tmux view

The tmux view shows worker activity in separate panes. This helps you:

- see each worker at once
- follow progress without losing context
- spot where a task is paused
- compare output across workers

If you are new to tmux, think of it as a screen split into sections. Each section shows one part of the job.
